# FREZZ LAUNDRY WEB APP - Frontend

Frontend multi-page berbasis HTML, CSS, dan Vanilla JavaScript.

## Halaman

- `login.html`
- `dashboard.html`
- `transaksi-baru.html`
- `daftar-cucian.html`
- `riwayat.html`
- `master-jenis.html`
- `users.html`

## Fitur

- Login JWT
- Sidebar role-based
- Dashboard owner dan kasir
- Input transaksi kiloan dan satuan
- Tabel cucian aktif
- Riwayat dengan pencarian
- CRUD master jenis
- CRUD users

## API Base URL

Frontend menggunakan:

`http://localhost:3000`
