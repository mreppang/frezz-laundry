# FREZZ LAUNDRY WEB APP

Full stack laundry management system berbasis:

- Frontend: HTML, CSS, Vanilla JavaScript
- Backend: Node.js + Express.js
- Database: MySQL (XAMPP / phpMyAdmin)
- Authentication: JWT
- Password Hash: bcrypt

## Folder

- `back end/` untuk REST API Express.js
- `front end/` untuk halaman web admin laundry
